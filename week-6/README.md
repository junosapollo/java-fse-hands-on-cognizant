# Week 6 — React HOL 1–8

The numbered applications correspond to React HOL documents 1 through 8. HOL 1–6 were already present and have been moved here; HOL 7 is `shoppingapp` and HOL 8 is `counterapp`.

Run one application with `npm install`, `npm test -- --watchAll=false`, or `npm run build` from that application directory. `scripts/verify-react.sh` checks all eight applications.
