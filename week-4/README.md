# Week 4 — Spring REST and JWT

`spring-learn` contains the XML/annotation Spring examples, country and
employee REST resources, validation/error handling, HTTP Basic login and a
modern stateless JWT filter using Spring Security 6. Authenticate with
`user`/`pwd` at `/authenticate`, then send the returned bearer token to the
protected REST endpoints.

Run `../../mvnw -Dmaven.repo.local=../../.m2/repository -pl week-4/spring-learn test`.
