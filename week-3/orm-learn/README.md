# Week 3 — Spring Data JPA and Hibernate

This module turns the three JPA/Hibernate hand-ons into one runnable project:
country derived queries, stock JPQL/native queries, payroll relationships and
fetch graphs, plus a quiz aggregate demonstrating cascading and orphan removal.
It uses an in-memory H2 database for repeatable tests and can be pointed at an
external database with normal Spring datasource properties.

Run `../../mvnw -Dmaven.repo.local=../../.m2/repository -pl week-3/orm-learn test`.
