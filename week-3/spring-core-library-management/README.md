# Week 3 — Spring Core and Maven

This project covers all nine Spring Core/Maven exercises: XML beans, setter and constructor injection, Maven configuration, component scanning, AOP timing and advice, and a testable Spring context. `applicationContext.xml`, `constructor-context.xml`, and `annotation-context.xml` make each configuration style explicit.

```bash
../../mvnw -pl week-3/spring-core-library-management test
```
