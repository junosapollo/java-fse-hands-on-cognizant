# Week 2 — JUnit, Mockito, Spring testing, and SLF4J

`java-testing-lab` is a self-contained Maven/Spring Boot project. It implements the basic and advanced JUnit exercises, Mockito exercises, Spring MockMvc/integration exercises, dependency-mocking exercises, and SLF4J logging exercises. H2 is used only for tests.

```bash
../../mvnw -pl week-2/java-testing-lab test
```

The PL/SQL exercises are kept separately in `../plsql-banking` because they require Oracle syntax and runtime semantics.
